# Simple Calculator (C Language)

## Description
A versatile C program developed as part of my **Computer Systems Engineering** studies at Al-Azhar University - Gaza. This application has evolved from a simple addition tool into a functional calculator that performs multiple arithmetic operations.

## Features
* **Language:** C.
* **IDE:** Code::Blocks.
* **Operations Supported:** Addition, Subtraction, Multiplication, and Division.
* **Clean Code:** Structured with functions and clear logic for better readability and maintainability.
* **User-Friendly Interface:** Clear prompts for input and formatted output of results.

## How to Run
1. Clone the repository to your local machine.
2. Open the `.cbp` file in Code::Blocks.
3. Build and run the project.
